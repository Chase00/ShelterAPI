# ShelterAPI

Animal shelter API created in PHP

### Mamp:
- Running on port **3306**

### PhpMyAdmin:
- Create a new database named **shelter**
  - import the **shelter.sql** file
- Set up a new user with all privileges
  - user: mamp
  - pass: mamp

### Setup:

```
composer update
```

### Run:
```
php -S localhost:8000
```
